# mod_hikashop_carousel (HikaShop Carousel)

Moduł Joomla wyświetlający karuzelę produktów HikaShop (UIkit slider).

## Instalacja
1. Wejdź w **Rozszerzenia  Zarządzanie  Instaluj**.
2. Wgraj ZIP z katalogu dist/.

## Konfiguracja
- **Tekst przycisku na dole sekcji**: puste = przycisk nie wyświetla się.
- **Pozycja menu (Itemid)  dokąd prowadzi przycisk**: pojawia się dopiero, gdy tekst przycisku nie jest pusty.

## Struktura paczki
- Kod modułu: services/, src/, 	mpl/, script.php, mod_hikashop_carousel.xml
- Assety: media/ (CSS/JS + joomla.asset.json)
